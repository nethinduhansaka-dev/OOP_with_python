# Python OOP Practice

This workspace contains small Python examples for learning Object-Oriented Programming (OOP) concepts.

## Files

- `main.py` - Entry script (if used for running examples together).
- `animal.py` - Class example related to animals.
- `car.py` - Class example related to cars.
- `student.py` - Class example related to students.
- `multiple_inheritance.py` - Basic multiple inheritance example.
- `multiple_inheritance_with_init.py` - Multiple inheritance with constructor (`__init__`) behavior.
- `multilevel_inheritance_with_init.py` - Multilevel inheritance with constructor chaining.
- `diamond_problem.py` - Demonstrates the diamond inheritance pattern and method resolution.

## Requirements

- Python 3.x

## Run a File

From this folder, run any script directly:

```bash
python main.py
```

or:

```bash
python diamond_problem.py
```

Use the same pattern for other files.

## Learning Focus

- Classes and objects
- Inheritance
- Constructor chaining
- Multiple inheritance
- Method Resolution Order (MRO)
